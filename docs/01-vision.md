# Plataforma de Cursos

## Objetivo

Desarrollar una plataforma web para la gestión de cursos online.

Inicialmente permitirá:

- Registro de usuarios.
- Compra e inscripción a cursos.
- Visualización de clases.
- Seguimiento del avance del estudiante.
- Gestión de pagos.
- Cuestionarios de evaluación.

Tecnologías elegidas:

- Python
- HTML
- CSS
- PostgreSQL
- Docker
- Visual Studio Code
